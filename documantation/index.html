A) About the plugin - top

IE Alert is a handy jQuery plugin that shows a well designed warning message to the visitor who's trying to view your site with an older version of Internet Explorer browser, this warning message will convince him to upgrade his browser to a newer version of Internet Explorer or to an another modern web browser.

This message will appear if your visitor is using an older version of Internet Explorer

B) What the plugin does - top

With the IE Alert plugin the visitors who are browsing your site with Internet Explorer IE8 (or lower versions) will see an warning message. This warning message tries to convince them to upgrade to a newer version of Internet Explorer or to an another modern web browser if they wish to continue on your site . By clicking the browser logos at the bottom a new window/tab will be opened where your visitor can download the current version of the chosen browser.
The warning message contains a title text (title text can be modified, see options) a basic text (this text can be modified, see options) and 5 images. These clickable images are logos of modern web browsers (Google Chrome,Firefox,Internet Explorer 9,Safari,Opera) when the visitor chooses a browser and clicks on the image a new window/tab will open where the visitor can download the up to date version of the chosen browser.

C) Getting started - top

Folder Structure

When you unzip the downloaded file the folder structure looks the following:

  • Documentation
  • ThePlugin
Documentation - The documentation to the plugin.
ThePlugin - In the Plugin folder you can find the following files:

  • iealert
  • iealert.js
  • iealert.min.js
iealert - In the iealert folder you can find a css file called style.css and two images iealertsprite.png and bg.png image.
iealert.js - This is the plugin.
iealert.min.js - This is a minified version of the plugin.


First step

The first step is to include some required files into your HTML document.

1.Include the latest jQuery library right after the closing </head> tag in your html document.

Download jQuery from jquery.com
How jQuery works

2.Include the minified IE Alert plugin (iealert.min.js) right after the already included jQuery Library.

3.Include the style.css CSS file (which is in the iealert folder) right after the already included IE Alert plugin. Note, that the style.css file must be in the same folder as the iealertsprite.png and bg.png images are.


A simple example how your HTML markup will look like if you include the required files:

<!DOCTYPE html>
<html>
 <head>
 <meta charset="UTF-8">
  <title>TITLE</title>

   <script src="http://code.jquery.com/jquery-1.7.2.js" type="text/javascript"></script>
   <script src="iealert.min.js" type="text/javascript"></script>
   <link rel="stylesheet" type="text/css" href="iealert/style.css" />

 </head>
 <body>
  
 </body>
</html>

D) Using the plugin - top

If you included all the required files (see Getting started > First step) you only need to "call" the plugin. We "call" the plugin with this line:

$("body").iealert();

Just copy this code and paste it into a jQuery $(document).ready() function. (more about the jQuery $(document).ready() function)

Example: 

$(document).ready(function() {
	$("body").iealert();
});

If you done everything exactly as it was written you can test your website with Internet Explorer 8,7,6 and you will see the warning message appear.

E) Options - top

The IE Alert plugin accepts 3 options: support, title, text

− The support option

The support option defines on which version of Internet Explorer will the warning message appear.
The support option accepts 3 parameters:

IE8 - The warning message will appear on Internet Explorer 8,7,6. By default it's set to IE8

Example codes:

		$("body").iealert({ support: "ie8" });

The warning message will appear on Internet Explorer 8,7,6


		$("body").iealert({ support: "ie7" });

The warning message will appear on Internet Explorer 7,6


		$("body").iealert({ support: "ie6" });

The warning message will appear only on Internet Explorer 6



− The title option

The title option specifies the title text, by default its: Did you know that your Internet Explorer is out of date ?. You can modify this text to whatever you want.

The title

example code:

		$("body").iealert({ title: "your browser is old!" });

The title text after the change.



− The text option

The text option specifies the text what is after the title text, by default its: To get the best possible experience using our site we recommend that you upgrade to a newer version of Internet Explorer .... You can modify this text to whatever you want.

The text

example code:

		$("body").iealert({ text: "Please, update your browser" });

The text after the change.


Note, the text parameter accepts HTML tags like the HTML font tag . So you can style the text like this:

		$("body").iealert({ text: "Please, <font color="red">update</font> your browser" });

F) Changing the font style - top

If you want to change the font styles completely you need to change some CSS properties in the style.css (it's in the iealert folder).
By default the: font family is Arial , font weight is bold and the font color is #333333
To change these styles open the style.css CSS file and search for #ie-alert-panel
To change the paragraphs font size search for #ie-alert-panel p the default is 14px
To change the titles font size search for #ie-alert-panel span the default is 18px
